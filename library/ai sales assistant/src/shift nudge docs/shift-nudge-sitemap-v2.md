## 🎯 Strategic Context

### Business Model Transformation
- **From:** Launch-based course sales ($1997 one-time, $399/mo payment plans)
- **To:** Application-only membership ($199/mo or $1997/year) + team plans
- **Goal:** Convert 600K+ annual visitors into qualified applicants and recurring members

### Platform Architecture
- **Marketing Site:** Static/CMS (Basehub) for speed, SEO, and conversion optimization
- **Course Delivery:** Circle for member authentication, community, and content delivery
- **Integration:** Clean handoff between marketing site and Circle platform

## 🔄 User Flow Documentation

### Individual Membership Flow
1. **Discovery:** Homepage or content consumption
2. **Interest:** Apply via `/apply` form
3. **Screening:** Application review (48-hour email response)
4. **Approval:** Email with secure `/join?key=` link
5. **Purchase:** Payment plan selection and Stripe checkout
6. **Onboarding:** `/welcome` page then redirect to Circle
7. **Delivery:** All course content, community, and live calls in Circle

### Team Flow A (Demo Request)
1. **Discovery:** Homepage or direct marketing
2. **Interest:** `/team` marketing page
3. **Inquiry:** Tally demo request form
4. **Sales Process:** Manual outreach and demo scheduling
5. **Purchase:** Custom enterprise sales process
6. **Setup:** Manual Circle team configuration

### Team Flow B (Direct Purchase)
1. **Discovery:** Homepage or direct marketing
2. **Interest:** `/team` marketing page
3. **Purchase:** `/team/join` seat selection and checkout
4. **Payment:** Stripe team billing setup
5. **Setup:** Automated Circle team access

### Lead Generation Flow
1. **Entry:** Homepage, social media, or content marketing
2. **Interest:** `/resources` hub exploration
3. **Capture:** Opt-in to specific lead magnets
4. **Nurture:** Email sequence with value delivery
5. **Conversion:** Eventually guided to membership application

## 🔧 Technical Integration Specifications

### Platform Stack
- **Frontend:** Static site generator with Basehub CMS
- **Authentication:** Circle handles all member authentication
- **Payments:** Stripe for both individual and team billing
- **Forms:** Tally for demo requests, custom forms for applications
- **Email:** Integration with existing email automation platform
- **Analytics:** Standard web analytics on marketing site

### Key Integrations
- **Circle API:** Member onboarding and access management
- **Stripe:** Payment processing and subscription management
- **Basehub CMS:** Content management for blog and resources
- **Email Platform:** Application workflow and member communication
- **URL Key System:** Secure application approval workflow

### Security Considerations
- URL key validation for approved applicant access
- Stripe handles all PCI compliance for payments
- Circle manages member authentication and access control
- Standard web security practices for marketing site
