# FloatPrompt for Web

Generate clean markdown files alongside HTML pages, making websites AI-readable.

> **Note:** This project is in active development. APIs may change.

## Quick Start

```bash
npx floatprompt generate ./dist
```

## What it does

- Creates `.md` file alongside each `.html` file
- Generates `llms.txt` site index
- Generates `llms-full.txt` with complete content
- Creates `/float/` dashboard for humans
