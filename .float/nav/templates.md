# Templates

The `templates/` folder — scaffolding templates for new FloatPrompt projects.

## Contents

| Item | Purpose |
|------|---------|
| **.float/** | Template FloatPrompt System structure |

## What Ships

When `npx floatprompt` runs, it copies this template structure to the target project. The template includes:

- `system.md` — Boot loader template
- `nav/` — Empty nav folder (populated during init)
