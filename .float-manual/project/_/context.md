# Project Context

> What floatprompt is and what matters.

## The Project

**FloatPrompt:** The invisible OS for AI.

A recursive context system that:
- Installs into any project (`.float/`)
- Maintains itself via agents
- Gives AI instant project understanding

## What We're Building

Context-compiler: TypeScript + agents that automatically maintain `.float/`.

```
AI orchestrates
├── TypeScript functions (scan, parse, write)
├── CLI commands (git, shell)
├── Buoys (parallel subagents)
└── Own cognition (judgment)
```

See `.float-manual/logs/` for locked decisions:
