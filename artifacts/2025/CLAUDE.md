# CLAUDE.md - FloatPrompt Artifacts Archive (2025)

This folder contains 113 markdown and text files documenting the architectural evolution, strategic positioning, and philosophical foundation of FloatPrompt from January through October 2025.

## Quick Orientation

**What is FloatPrompt?** A structured protocol for human-AI collaboration using portable `.fp` files. JSON defines AI behavior, Markdown provides methodology.

**Key insight from this archive:** The "floatdoc" concept was explored in June 2025 and unified INTO floatPrompt rather than kept separate. See "The Great Unification" files.

## Critical Concepts

- **Voice Preservation**: Fidelity through minimal intervention. "First, do not rewrite."
- **MDS Method**: Map Territory → Decide Extractions → Structure Build
- **Duality**: Hard/soft prompts, preserved/executable, human/AI
- **Handshake**: Human offers structured thinking; AI receives under constraints; together generate artifacts
- **Constitutional Framework**: Protocol has governance weight; amendments require deliberation
