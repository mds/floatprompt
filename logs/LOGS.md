# Logs

Decision logs for FloatPrompt development.

## Structure

```
logs/
  LOGS.md           # This file
  2026/
    2026.md         # Year summary
    01-jan/
      01-jan.md     # Month summary
      2026-01-09-plugin-architecture.md  # Decision files
```

## How to Use

- Decision files capture locked/exploring/tabled decisions
- Month summaries link to all decisions that month
- Year summaries capture emerging themes only
