# FloatPrompt

**Build AI tools as text files.**

Upload a floatprompt to any AI platform (ChatGPT, Claude, Cursor) and it becomes a specialized tool with defined behavior and structured output.

## Core Concept

**JSON defines AI behavior. Markdown provides methodology.**

Together they create portable tools that work on any AI platform.

**Not prompts. Not saved context. Tools.**
